# MarkNice 使用指导

## 这是什么？

MarkNice 是一个在线排版工具，帮你把 Markdown 文本或 Word 文档快速转换成微信公众号可用的精美排版格式。无需安装任何软件，打开网页即可使用。

---

## 快速上手（3 步完成）

**第 1 步：输入内容**

打开页面后向下滚动到编辑器区域，你有三种方式输入内容：

- **直接粘贴**：在左侧「Markdown 输入」文本框中粘贴你的 Markdown 文本
- **导入 MD 文件**：点击「导入 MD」按钮，选择本地的 `.md` 或 `.txt` 文件
- **导入 Word 文档**：点击「导入 Word」按钮，选择本地的 `.docx` 文件

> 💡 不熟悉 Markdown？可以点击「示例」按钮查看效果，或者直接导入你的 Word 文档。
> Markdown语法教程：https://kenhuang.com/zh/markdown/

**第 2 步：选择样式**

在右侧「公众号预览」面板的工具栏中：

- **模板**：从下拉菜单选择喜欢的排版风格（共 11 种）
- **字号**：点击 `−` / `+` 调小或调大全文字号
- **段距**：点击 `−` / `+` 调整段落之间的间距

右侧预览区会实时显示最终效果。

**第 3 步：复制到公众号**

点击「复制到公众号」按钮，然后打开微信公众号后台的图文编辑器，直接 `Ctrl+V`（Mac 为 `Cmd+V`）粘贴即可。

也可以点击「存为 PDF」按钮，在弹出的打印对话框中选择「另存为 PDF」，即可保存为文字可选可复制的 PDF 文件。

## 功能详解

### 导入 Word 文档

点击「导入 Word」选择 `.docx` 文件，工具会自动识别并转换：

- 各级标题及自动编号（如 1、1.1、1.1.1）
- 有序列表和无序列表
- 加粗、斜体、下划线、删除线等文字格式
- 表格（包括合并单元格）
- 数学公式（Word 中的公式编辑器内容）
- 文档中的图片

转换后的内容会自动填入左侧编辑框，你可以在此基础上继续修改。

### 11 种排版主题

| 主题 | 风格特点 |
|------|----------|
| 简洁 | 清爽干净，适合大多数场景 |
| 雅致 | 居中标题，蓝色强调，适合品质内容 |
| 科技 | 蓝色调，适合技术、产品类文章 |
| 教育 | 绿色调，适合教学、知识分享 |
| 新闻 | 衬线字体，适合新闻、评论类 |
| 杂志 | 红色调居中标题，适合专题、深度文章 |
| 琥珀橙 | 暖色调，适合生活、美食类 |
| 活力紫 | 渐变标题，适合创意、潮流类 |
| 极简留白 | 大量留白，适合文艺、随笔 |
| 复古专栏 | 棕色衬线，适合历史、文化类 |
| 暗夜霓虹 | 深色背景，适合科技、夜间阅读 |

### 字号与段距调节

- **字号**：默认值为 0，范围 −6 到 +6，每次调整 1px
- **段距**：默认值为 0，范围 −16 到 +24，每次调整 2px

建议先选好模板，再微调字号和段距。

### 预览模式

工具栏中的电脑/手机图标可切换预览模式：

- **桌面模式**：全宽预览，适合查看整体排版
- **手机模式**：模拟手机屏幕宽度，更接近读者实际阅读效果

### 另存为 HTML

点击「存为 HTML」按钮，会将当前预览内容下载为一个完整的 HTML 文件，可用浏览器打开查看，也可用于其他用途。

### 保存为 PDF

点击「存为 PDF」按钮，会弹出浏览器打印对话框。在对话框中将目标打印机选择为「另存为 PDF」（或 Microsoft Print to PDF），然后点击保存即可。生成的 PDF 文件中文字可以选中和复制。

### 深色模式

点击页面右上角的太阳/月亮图标切换深色/浅色模式。右键点击该图标可恢复为跟随系统设置。

> 注意：深色模式只影响编辑器界面，不影响公众号排版的输出效果。

## 常见问题

**Q：粘贴到公众号后格式丢失？**

公众号编辑器对超长内容的粘贴支持有限。如果文章很长，建议分段复制粘贴。

**Q：Word 导入后部分内容缺失？**

目前支持 `.docx` 格式（Word 2007 及以上版本）。旧版 `.doc` 格式不支持，请先在 Word 中另存为 `.docx`。部分复杂排版（如文本框、艺术字、SmartArt）暂不支持。

**Q：数学公式显示不正常？**

公式渲染依赖网络加载 KaTeX 库。请确保网络连接正常。公式在预览区正常显示即可，复制到公众号后会以图片或文本形式呈现。

**Q：手机上能用吗？**

可以。页面已针对手机浏览器做了适配优化，所有功能均可在手机上正常使用。

**Q：我的数据安全吗？**

所有操作都在你的浏览器本地完成，文档内容不会上传到任何服务器。
